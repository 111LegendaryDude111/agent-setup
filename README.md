# Codex setup files

Запустите команду из корня репозитория, куда нужно добавить эти файлы.

```bash
curl -L https://codeload.github.com/111LegendaryDude111/agent-setup/tar.gz/refs/heads/main | tar -xz --strip-components=1
```

Команда скачает файлы из корня репозитория и положит их в текущую папку.
