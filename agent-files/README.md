# Документация для агентов

Этот каталог хранит поддерживающие файлы агента: долгосрочный контекст, короткий handoff, подробный task log, список skills и инструкции subagents.

Основной контракт работы агента находится в корне репозитория: `AGENTS.md`.

## Файлы

| Файл | Описание |
| --- | --- |
| `AGENT_HANDOFF.md` | Короткий volatile-файл для продолжения незавершенной задачи. Обычно `status: clear`; не хранит историю, команды или подробные отчеты. |
| `AGENT_TASK_LOG.md` | Подробный хронологический журнал действий агента. Его читают только когда нужен контекст прошлых действий. |
| `CONTEXT.md` | Долгосрочный контекст проекта: устойчивые решения, ограничения, архитектурные договоренности, known issues и важные изменения. |
| `SKILLS.md` | Рекомендуемые skills для установки и использования: базовые, coding, architecture, documentation, audit и optional tools. |
| `README.md` | Краткая карта документации и назначение файлов в этом каталоге. |

## Дополнительные файлы

1. `README.md`
2. `AGENTS.md`
